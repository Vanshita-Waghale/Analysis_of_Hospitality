
### 🔍 Hospitality Data Analysis Project using Python

This project is part of the **Codebasics Data Analyst Bootcamp**, where I analyzed real-world booking data from **AtliQ Grands**, a luxury 5-star hotel chain operating across four major Indian cities.
The goal was to uncover insights through **exploratory data analysis (EDA)** to support smarter, data-driven decisions and improve business performance.

**📊 Project Highlights:**

* Analyzed data across **7 properties** and **4 room categories** using Python (Pandas, NumPy, Matplotlib).
* Explored trends in bookings, room popularity, and occupancy using datasets such as:

  * `dim_date`: time-based insights (weekends, weekdays, seasonality)
  * `dim_hotels`: property-specific analysis
  * `dim_rooms`: room category trends
  * `fact_aggregated_bookings`: booking volumes and capacity
* Delivered insights to support pricing strategy, occupancy maximization, and marketing optimization.

**🛠️ Tools & Technologies:**
Python, Pandas, NumPy, Matplotlib, Jupyter Notebook
